# Big decision
Yeah I decided to move from Gatsby to Jekyll to build my Github blog on more React-friendly environment.

# Steps
Install the cli tool

    ```
    npm install --global gatsby-cli
    ```
    
Initialize a new site

    ```
    gatsby new gatsby-site
    ```
