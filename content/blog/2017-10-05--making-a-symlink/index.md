## [How to](https://stackoverflow.com/questions/1951742/how-to-symlink-a-file-in-linux)
Create a symlink
```
ln -s /path/to/file /path/to/symlink
```
Create/update a symlink
```
ln -sf /path/to/file /path/to/symlink
```
